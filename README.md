# Реализация BitSet на Java

Данная структура, которая является аналогом `java.utils.BitSet` в Java и `std::bitset` в С++, создана в образовательных целях как демонстрация конкретного подхода и для дальнейшего использования и кастомизаций в проектах, которые требуют более сложные и специфические структуры данных.

## О структуре
Bitset - структура, позволящая представлять большой массив бит как одно число, записанное в бинарном виде в little-endian форме, и эффективно производить с ним битовые операции. Чаще всего каждые 2^64 бит хранятся в одном 64-х битном числе, что делает работу с памятью эффективнее и позволяет применять машинные инструкции.
## Особенности
- Наличие всех необходимых методов для работы с битами
- В отличии от `java.utils.BitSet` имеет эффективно реализованные `clz`, `ctz`, `popCount`, `findFirst`(в 3 раза быстрее аналога из `utils`), `findLast`
- Константный размер структуры


## Методы

### `set(int index, bool value)`
Устанавливает значение бита по заданному индексу.
- **Параметры**:
  - `index` – позиция бита (начиная с 0).
  - `value` – `true` для установки бита в 1, `false` для сброса в 0.
- **Возвращает**: void.

### `get(int index)`
Возвращает текущее значение бита по индексу.
- **Параметры**: `index` – позиция бита.
- **Возвращает**: `1`, если бит установлен, иначе `0`.

### `popCount()`
Возвращает количество единичных битов в множестве.
- **Возвращает**: число >= 0.

### `clz()`
Подсчитывает количество ведущих нулей в little-endian записи.
- **Возвращает**: целое число >= 0

### `ctz()`
Подсчитывает количество хвостовых нулей.
- **Возвращает**: целое число >= 0
  
### `findFirst()`
Находит индекс первого младшего установленного бита.
- **Возвращает**: индекс бита или `-1`.

### `findLast()`
Находит индекс последнего старшего установленного бита.
- **Возвращает**: индекс или `-1`.

### `findNext(int from)`
Ищет следующий установленный бит, начиная с позиции `from` исключительно 
- **Параметры**: `from` – начальная позиция поиска.
- **Возвращает**: индекс найденного бита или размер множества, если достигнут конец.

### `not()`
Выполняет побитовое отрицание текущего множества.  
- **Возвращает**: void.

### `or(Bitset bs)`, `and(Bitset bs)`, `xor(Bitset bs)`
Побитовое ИЛИ, И, ЛИБО соответсвтенно
- **Параметры**: другое битовое множество.
- **Возвращает**: void.
